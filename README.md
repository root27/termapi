## Http Inspector via terminal

Simple terminal http inspector in golang

## Key Mappings

You can find key shortcuts in the table below.

| Key | Function |
| --- | ---       |
| `Ctrl+S`| `Send request`|
| `Ctrl+R`| `Send new request`|
| `Ctrl+E`| `Save response`|
| `Ctrl+C`| `Quit`|
| `Tab`| `Next view`|
| `Esc`| `Close the save modal`|
